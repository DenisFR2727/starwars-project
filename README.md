# Star Wars Project

React + TypeScript + Vite проект для роботи з Star Wars API.

## Встановлення

```bash
npm install
```

## Запуск проекту

```bash
npm run dev
```

## Тестування

Проект використовує Vitest для unit тестування кастомних хуків.

### Запуск тестів

```bash
# Запустити всі тести
npm test

# Запустити тести в режимі watch
npm test

# Запустити тести з UI
npm run test:ui

# Запустити тести з покриттям коду
npm run test:coverage
```

### Написані тести

1. **useFetchHeroes.test.ts** - Тести для хука отримання списку героїв

   - Перевірка успішного завантаження
   - Обробка помилок API
   - Розрахунок загальної кількості сторінок
   - Рефетч при зміні сторінки
   - Обробка порожніх результатів

2. **useBuildNodes.test.tsx** - Тести для хука створення графа

   - Створення ноди героя
   - Створення нод фільмів
   - Створення нод космічних кораблів
   - Створення зв'язків між нодами
   - Адаптивне позиціонування для мобільних пристроїв
   - Фільтрація космічних кораблів

3. **useFetchDetailsHero.test.ts** - Тести для хука отримання деталей героя
   - Успішне завантаження даних
   - Обробка відсутнього ID
   - Обробка порожніх масивів фільмів/кораблів
   - Обробка помилок API
   - Рефетч при зміні ID
